Project 4: iOS Metronome
05-4/633 C
SSUI Mobile Lab

This is a basic implementation of a metronome that allows you to specify a tempo in beats per minute, and a number of beats per measure.

Instructions
============

 *  Swipe down to access settings
 *  Tap and hold down your finger (long press) to stop the metronome
 *  The metronome will stop and reset if you begin to swipe down to access settings
